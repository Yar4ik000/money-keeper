# Money Keeper

Личный финансовый трекер для Android. Работает полностью офлайн — данные хранятся только на устройстве, зашифрованные мастер-паролем.

---

## Возможности

- **Счета** — карты, наличные, вклады, накопительные и инвестиционные счета с иконками и цветовой маркировкой
- **Транзакции** — доходы, расходы, переводы между счетами, повторяющиеся операции
- **Категории** — двухуровневое дерево с иконками, настраиваемое под себя
- **Бюджеты** — лимиты расходов по категориям и счетам на месяц или неделю
- **Аналитика** — круговые и столбчатые диаграммы, история с фильтрами и мультиселектом
- **Прогнозы** — расчёт баланса на выбранную дату с учётом вкладов и повторяющихся операций
- **Вклады** — простые и сложные проценты, капитализация, уведомления об истечении срока
- **Резервные копии** — зашифрованный `.mkbak`-архив; экспорт в CSV для Excel
- **Безопасность** — шифрование AES-256 (SQLCipher), Argon2id KDF, биометрическая разблокировка, автоблокировка

## Установка

Приложение распространяется APK-файлом, минуя Google Play.

### 1. Разреши установку из неизвестных источников

**Android 8+:** Настройки → Приложения → [файловый менеджер / браузер] → Установка неизвестных приложений → Разрешить.

**Android 7 и ниже:** Настройки → Безопасность → Неизвестные источники → включить.

### 2. Установи APK

Передай файл `app-release.apk` на телефон (USB, Telegram, email) и открой через файловый менеджер → **Установить**.

### 3. Создай мастер-пароль

При первом запуске приложение попросит придумать мастер-пароль. Это единственный ключ к твоим данным — **нигде не хранится и не восстанавливается**. Запиши его в менеджер паролей (Bitwarden, KeePassXC).

### Обновление

Просто установи новый APK поверх старого — данные сохранятся.

> Полная инструкция по установке, обновлению, переносу на новый телефон и частые вопросы: **[docs/INSTALL_GUIDE.md](docs/INSTALL_GUIDE.md)**

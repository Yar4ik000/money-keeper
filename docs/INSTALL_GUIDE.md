# MoneyKeeper — Инструкция по установке и обновлению

Приложение распространяется APK-файлом напрямую (не через Google Play).

---

## Первая установка

### 1. Разреши установку из неизвестных источников

На **Android 8 и выше** разрешение выдаётся под конкретное приложение:

1. Открой **Настройки → Приложения** (или «Управление приложениями»).
2. Найди **Файловый менеджер** (или браузер, через который будешь открывать APK).
3. Тапни по нему → **Установка неизвестных приложений** → включи переключатель.

На **Android 7 и ниже**:
1. **Настройки → Безопасность → Неизвестные источники** → включи.

### 2. Скачай и установи APK

1. Скинь файл `app-release.apk` на телефон (через Telegram, email, USB — как удобно).
2. Открой его через файловый менеджер.
3. Нажми **Установить**.

### 3. Первый запуск — создание пароля

При первом запуске приложение попросит создать **мастер-пароль**. Это главный
и единственный ключ ко всем твоим данным.

**Важно:**
- Пароль нигде не хранится и не восстанавливается — запомни его или запиши
  в менеджер паролей (Bitwarden, KeePassXC и т.д.).
- Если забудешь пароль — данные будут безвозвратно потеряны. Приложение сотрёт
  базу и начнёт с чистого листа.
- Рекомендуется периодически делать зашифрованный бэкап через
  **Настройки → Резервная копия → Создать бэкап** и хранить файл `.mkbak`
  в облаке или отправлять себе в Telegram-избранное.

## Обновление

Когда выйдет новая версия — просто **поставь новый APK поверх старого**:

1. Скачай новый `app-release.apk`.
2. Открой и нажми **Обновить** (Android покажет именно эту кнопку, если подпись совпадает).
3. После установки — разблокируй приложение своим паролем.
4. Все данные (счета, транзакции, вклады, настройки) сохраняются автоматически.

**Обновление не трогает твои данные** — база данных и ключи шифрования остаются
на устройстве нетронутыми.

## Резервные копии (важно!)

### Создание бэкапа
**Настройки → Резервная копия → Создать бэкап**

Файл `.mkbak` — зашифрованный архив, защищённый твоим мастер-паролем.
Без пароля прочитать его невозможно.

Куда сохранять:
- Telegram «Избранное» (удобно и надёжно)
- Google Drive / Яндекс.Диск
- На компьютер

### Восстановление из бэкапа
**Настройки → Резервная копия → Восстановить бэкап**

Выбери `.mkbak`-файл и введи мастер-пароль, который был **на момент создания** бэкапа.

### Перенос на новый телефон
1. Сделай бэкап на старом телефоне.
2. Установи приложение на новый телефон (первая установка выше).
3. При первом запуске создай новый мастер-пароль.
4. Восстанови бэкап и введи **старый** мастер-пароль (тот, которым шифровался бэкап).

## Factory reset / смена телефона без бэкапа

Если телефон сломался или был сброшен до заводских настроек **без предварительного бэкапа**
— данные восстановить невозможно. Ключи шифрования хранятся в защищённом хранилище
операционной системы и удаляются при сбросе.

**Вывод:** делай бэкап регулярно. Одного раза в неделю достаточно.

## Часто задаваемые вопросы

**Можно ли установить на несколько телефонов?**
Да. Используй бэкап для синхронизации: создай бэкап на одном устройстве,
восстанови на другом. Автоматической синхронизации нет — это осознанное решение
для максимальной приватности (данные не уходят на сторонние серверы).

**Приложение просит разрешения на уведомления — зачем?**
Для напоминаний о плановых расходах и уведомлений об окончании вкладов.
Можно отклонить — основной функционал работает без уведомлений.

**Что значит «биометрия» в настройках?**
Дополнительный способ разблокировки — вместо ввода пароля при каждом открытии.
Мастер-пароль при этом не отменяется: он нужен для создания и восстановления бэкапов.

**Почему нет синхронизации с облаком / веб-версии?**
Приложение хранит данные только локально. Это гарантирует, что твои финансовые данные
не попадут на сторонние серверы. Бэкап — единственный механизм переноса данных.
